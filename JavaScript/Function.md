###1. Anonymous function
Anonymous function doesn't have a name but, it is a function.

```
// Anonymous function
var func = function() { 	
	alert("Hello");
};

// We can see contents of the func
alert(func);

// It looks like string but, it is a function type variable
alert(typeof(func));

```
